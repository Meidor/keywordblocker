• Here you can add channels that should be blocked by Keyword Blocker. Normally Keyword Blocker matches the whole name of the channel but if you use the wildcard option than partial matches are also blocked.

    For example if you add Test as a blocked channel only the channnel named Test will be blocked. If you enable the wildcard option Test Channel, Test Tube etc will also be blocked.

    Channel blocking is case insensitive and regular expressions are supported for advanced users.
